## Step 2 - Modules ([Full Documentation)](/modules)

<Steps>
  <Step title="Most importantly, run the configuration command /config.">
    <Frame>
      ![Image](/images/image-18.png)
    </Frame>
  </Step>
  <Step title="Config Menu & Click on Modules">
    <Frame>
      ![Image](/images/image-19.png)
    </Frame>

    Click on the module select option within in the Config Menu.
  </Step>
  <Step title="Selecting the modules">
    <Frame>
      ![Image](/images/image-20.png)
    </Frame>

    Click on the modules dropdown option, for the modules you want to select them.
  </Step>
  <Step title="Click anywhere else.">
    <Frame>
      ![Image](/images/image-21.png)
    </Frame>

    Now click anywhere else but the dropdown menu, it should prompt a `Modules Saved` popup.

    The module is now the module is enabled!
  </Step>
</Steps>
