---
title: Forums
description: Automatically send a message when a forum post is created
---

The **forums** module automatically sends a message when a new forum post is created by a user. You can configure it
by selecting the module in `/config` and then selecting Manage Forums. From there you can create a new forum message,
edit an existing one or remove a forum message.

## Configuration Options

    <ParamField path="color" type="blue/red/gray/green" default="gray">
        What color is the button?
    </ParamField>
    <ParamField path="command" type="string" required>
        Custom command to execute when pressed
    </ParamField>
</ParamField>
