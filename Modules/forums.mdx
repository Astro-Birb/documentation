---
title: Forums
description: A way to automatically send a message when a forum post is created.
---

> Use the **Forums** module to automatically send a customizable embed message whenever a new forum post is created in your Discord server.

<AccordionGroup>
  <Accordion title="Customize Embed">
    Allows full customization of the forum embed to your liking. You can modify the following fields:
    - Button
    - Title
    - Description
    - Thumbnail
    - Image
    - Author
    - Color
    - Fields
    - Ping
    - Forums Channel
  </Accordion>
</AccordionGroup>
